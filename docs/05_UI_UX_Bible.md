# Chapter 5 — UI/UX Bible

## Purpose

Define the principle that UI is not a neutral overlay: every major interface belongs to a companion.

## UX Principles

- Avoid generic management menus.
- Make information feel delivered by a person with a point of view.
- Show consequences visually and narratively, not only numerically.
- Use interface tone, motion, and layout to reinforce character identity.
