# Dream Palace Vision Bible

## Purpose

Define the creative north star for Dream Palace: a character-driven fantasy kingdom management RPG where the kingdom is built through companions, consequences, and a living world.

## Core Pillars

- Characters first: every system should deepen emotional connection with the seven companions.
- Living kingdom: cities, people, factions, resources, and politics continue to evolve.
- Decisions matter: important choices should create visible world consequences.
- UI is character: major interfaces should feel owned by people, not generic menus.

## Project Promise

We are not building only a kingdom.

We are building a world worth remembering.
