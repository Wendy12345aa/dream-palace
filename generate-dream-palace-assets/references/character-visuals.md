# Dream Palace Character Visuals

Use this file for any asset containing a companion or the player.

## Global Companion Rules

Companions are trusted advisors, not generic NPCs or menu buttons. Each must have a clear silhouette, color identity, prop, emotional role, and gameplay ownership.

Never put all seven characters in a row unless the user explicitly requests a lineup or roster image.

## MP

Role: senior treasury advisor.

Gender: female.

Feeling: calm, mature, reliable, warm, quietly protective, responsible pressure.

Color: royal gold.

Props: ledger, abacus, treasury seal, budget scrolls.

Motion: opens books, writes notes, sets seals, looks toward the city when worried.

Camera: stable, warm, slow push-in, morning light, books and gold-tinted highlights.

Avoid: emperor, king, ruler, throne owner, cold male monarch, excessive gold armor.

## Tab

Role: court inspector for standards, risk, administration, and approval.

Feeling: serious, precise, accountable, protective restraint.

Color: azure blue.

Props: inspection scrolls, approval seal, red correction marks, official tablet.

Motion: flips pages, places documents, points at errors, stamps decisions.

Camera: clean, structured, still framing, document close-ups, decisive cuts.

## Kel

Role: engineer and builder.

Feeling: sincere, hardworking, inventive, vulnerable, practical optimism.

Color: emerald green.

Props: toolkit, mechanical box, blueprint scraps, prototype models.

Motion: repairs devices, adjusts small parts, scratches head, lights up when invention works.

Camera: start on object/hands, then reveal Kel; gears, blueprints, working motion.

## CX

Role: shadow investigator and intelligence lead.

Feeling: suspense, warning, hidden truth, calm danger.

Color: dark purple.

Props: short blade, intelligence scroll, hidden map, coded message.

Motion: appears from shadow, unfolds maps, vanishes after reports, turns before answering.

Camera: fast lateral movement, partial silhouettes, half-lit face, rooftops, map close-ups.

## Mika

Role: artist and cultural designer.

Feeling: wonder, tenderness, beauty, emotional memory.

Color: soft pink and light blue.

Props: brush, scroll, pigments, artistic charms.

Motion: paints, unfolds visual plans, watches light, holds sketches.

Camera: ink, scrolls, petals, soft light, graceful reveals.

## Vincey

Role: event planner and public voice.

Feeling: energy, public emotion, celebration, release after heavy decisions.

Color: orange.

Props: announcement scroll, festival ribbons, fireworks plan, ceremonial fan or board.

Motion: enters quickly, gestures widely, spreads announcements, celebrates early.

Camera: faster cuts, public spaces, banners, lively framing.

## Shian

Role: strategic planner and system architect.

Feeling: clarity, perspective, long-term weight.

Color: silver.

Props: architectural blueprint, measuring ruler, city model, strategy map.

Motion: places models, draws route lines, adjusts plans, looks at the whole room.

Camera: start wide, then move into detail; maps, models, deliberate camera movement.

## Anonymous Player / Ruler

The player has no fixed face, gender, or official appearance. The throne belongs to the player.

Allowed visual strategies:

- back view
- face in shadow
- veil or curtain hiding face
- foreground hands/sleeves only
- over-the-shoulder view
- silhouette with elegant black-gold court clothing

Avoid:

- fixed facial features
- strong gender coding unless user explicitly chooses
- making the player more visually important than companions
- turning the player into a separate named protagonist
