# Dream Palace Web Demo

This folder contains the first playable prototype for **Dream Palace: The First Morning Court**.

The goal is not to build the full game.

The goal is to prove the first emotional loop:

```text
Morning Court
-> MP report
-> player decision
-> companion reaction
-> visible kingdom change
```

## How To Run

Open `index.html` in a browser.

No build step is required for the first prototype.

## Current Scope

Included:

- Morning Court scene
- MP first report
- one player decision
- Tab reaction
- Kel construction response
- small kingdom state update

Excluded:

- real AI
- backend
- save system
- full asset pipeline
- full Demo Bible runtime
- all seven active companions

## Design Rule

If the prototype starts to feel like a dashboard, stop and redesign the interaction around a companion.
